# Atajos de Teclado de ComfyUI

| Atajo | Acción |
|-------|--------|
| Ctrl + Enter | Encolar generación actual |
| Ctrl + Shift + Enter | Encolar como primera |
| Ctrl + Alt + Enter | Cancelar generación |
| Ctrl + Z / Y | Deshacer / Rehacer |
| Ctrl + S | Guardar workflow |
| Ctrl + O | Cargar workflow |
| Ctrl + A | Seleccionar todos los nodos |
| Alt + C | Colapsar/expandir nodos seleccionados |
| Ctrl + M | Silenciar/activar nodos |
| Ctrl + B | Bypass nodos |
| Delete / Backspace | Eliminar nodos seleccionados |
| Ctrl + Backspace | Limpiar el grafo |
| Space + Drag | Mover el canvas |
| Ctrl + C / V | Copiar y pegar nodos |
| Ctrl + C + Shift + V | Pegar manteniendo conexiones |
| Ctrl + D | Cargar grafo por defecto |
| Alt + + / - | Zoom in / Zoom out |
| P | Pin/Unpin nodos seleccionados |
| Ctrl + G | Agrupar nodos seleccionados |
| Q | Mostrar/ocultar la cola |
| H | Mostrar/ocultar el historial |
| R | Refrescar el grafo |
| F | Mostrar/ocultar menú |
| . | Ajustar vista a selección |
| Double-Click LMB | Búsqueda rápida de nodos |
| Ctrl + Alt + LMB | Desconectar todos los cables del slot |

## Sintaxis de Prompts

(palabra:1.2) - Énfasis aumentado (default: 1.1)
(palabra:0.8) - Énfasis reducido
{día|noche} - Prompt dinámico/wildcard
embedding:nombre - Usar textual inversion
BREAK - Separador de condicionamiento (SDXL)

## Notas para macOS

Ctrl puede reemplazarse por Cmd en macOS.
